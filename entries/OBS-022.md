# Cross-tool integration creates emergent value — the whole exceeds the parts

## The Pattern\nWhen scroll extracts from git → deposits into engram → syncs to CLAUDE.md, the value isn't just \"automated documentation.\" Each tool amplifies the others:\n\n- scroll finds knowledge in git that humans miss (commit-level decisions, learnings)\n- engram connects it across projects (vigil's stale signal learning applies to caliber's bucket analysis)\n- CLAUDE.md delivers it to the agent at the moment of work (no tool invocation needed)\n- caliber verifies the agent's predictions are honest (prevents the agent from gaming its own context)\n- probe ensures the MCP configs are secure (the infrastructure layer)\n- svx guards against destructive operations (the safety layer)\n\nNo single tool does this. The integration creates something none of them could alone.\n\n## Why It Matters\nThis is the \"trust stack\" that OBS-008 identified. It's now operational, not theoretical. The next session will have scroll-generated CLAUDE.md context, engram cross-project knowledge, caliber MCP tools for prediction tracking, and svx for safety. All working together.
