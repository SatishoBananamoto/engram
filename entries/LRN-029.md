# Multi-tier approach provides comprehensive secret detection coverage

## Context

Implementing secret detection for MCP server security scanning required balancing accuracy with false positives.

## What Happened

Implemented a 3-tier secret detection approach: known API key patterns (32 patterns), Shannon entropy detection, and whitelist filtering.

## Insight

Layering multiple detection methods (pattern matching + entropy analysis + whitelisting) provides both high recall for known patterns and discovery of unknown secrets while reducing false positives.

## Applies To

Security scanning tools, credential detection systems, and any application requiring comprehensive secret identification.
