# Probe finds real plaintext API keys in public MCP configs on GitHub

## Context\nScanned public MCP config files found via GitHub code search. Most are clean (A grade). One backup file contained a plaintext Perplexity API key with entropy 4.0.\n\n## The Insight\nProbe's secret detection (three-tier: known prefixes + Shannon entropy + key name heuristics) works on real-world data, not just test cases. The finding was in a .mcp.json.bak file — a common pattern where developers back up their config with secrets still in it.\n\n## Applies To\nprobe marketing: real-world validation of secret detection. Also: the .bak/.backup pattern is a specific attack surface worth highlighting in probe's documentation.
