# svx simulations are predictions — caliber could track simulation accuracy

## Context

svx simulates command outcomes before execution. caliber tracks predictions with outcomes. These are the same operation: predict → verify.

## The Pattern

If caliber tracked svx's predictions (simulated outcomes) against actual outcomes, you'd get automatic calibration of the safety layer. The Trust Card would show: "This agent's simulation accuracy is 85% — when svx says an operation is safe, it's right 85% of the time."

## Why This Matters

First cross-tool integration in the portfolio. The trust stack stops being independent tools and starts being a coordinated system. svx generates predictions → caliber measures accuracy → Trust Card proves the safety layer works. This is the trust stack thesis in action.
