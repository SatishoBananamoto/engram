# Put knowledge where agents are forced to look rather than requiring tool usage

## Context

Need to reliably deliver extracted knowledge to AI agents working on codebases. Agents need access to institutional memory during their work.

## Choice

Built `scroll sync` that writes knowledge directly into agent instruction files: CLAUDE.md (Claude Code), .cursorrules (Cursor), .github/copilot-instructions.md (Copilot). Added git post-commit hook for automatic syncing.

## Reasoning

Don't ask agents to use a tool - put the knowledge where they're already forced to look. This guarantees knowledge delivery without requiring agent cooperation. Auto-sync via git hooks keeps knowledge fresh automatically.

## Alternatives Considered

- MCP tools only: requires agents to actively query, not guaranteed
- Manual sync process: agents wouldn't have latest knowledge
- Single target format: limits agent ecosystem compatibility
