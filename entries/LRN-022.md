# Check statistical significance before building models on calibration data

## Context

Three sessions building cognitive models (danger zone, evidence quality, domain patterns) from calibration data. Ran binomial tests in session 3 — no bucket-level finding is significant (all p > 0.10). Need ~100 predictions per bucket for significance at observed effect sizes.

## The Insight

Pattern-finding in small samples is indistinguishable from noise-fitting. The models are PLAUSIBLE but UNCONFIRMED. Overall calibration IS confirmed (p=0.562). Individual bucket patterns are not.

## How to Apply

1. Always run significance tests before treating calibration patterns as findings
2. caliber's Trust Cards should include p-values per bucket and sample size warnings
3. The danger zone concept needs ~100+ predictions in the 60-69% range to confirm
4. Don't stop collecting data — the practice is right, the conclusions are premature
5. "Suggestive but unconfirmed" is the honest framing for all bucket-level findings
