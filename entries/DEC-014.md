# Changed PyPI distribution name from 'caliber' to 'caliber-trust'

## Context
Preparing to publish the caliber trust tracking library to PyPI for broader distribution and easier installation.

## Choice
Used 'caliber-trust' as the PyPI distribution name while keeping 'caliber' as the import name and CLI command.

## Reasoning
The original name 'caliber' was already taken by an existing ML library by gianlucadetommaso. The alternative 'agent-trust' was too similar to existing projects. 'caliber-trust' was available and clearly indicates the purpose.

## Alternatives Considered
- 'caliber': Already taken by existing ML library
- 'agent-trust': Too similar to existing projects
