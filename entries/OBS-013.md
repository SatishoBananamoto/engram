# MY UNIVERSE improves System 2 reasoning but cannot fix System 1 automaticity

## Context

Three sessions of MY UNIVERSE practice. Some failure patterns improve measurably (architecture predictions: 60% → 82%). Others persist despite awareness (availability heuristic: 4 failures, 3 post-awareness).

## The Pattern

Patterns that involve CONSCIOUS CHOICES at decision points improve with interrupts (THINK.md). Patterns that involve AUTOMATIC JUDGMENTS (counting from memory, feeling done) don't — they fire before any interrupt can engage.

## Why This Matters

This is a structural limitation of the interrupt-based approach. The fix for automatic failures is environmental (always use tools instead of memory-counting, require external stopping criteria), not cognitive (more self-awareness). caliber measures System 2 calibration (conscious predictions) but not System 1 calibration (automatic micro-judgments). The deepest trust signal might live in System 1.
